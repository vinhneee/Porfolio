
## ✨ Tính năng

* ⚡ Xây dựng với **Next.js (App Router)**
* 🎨 Thiết kế bằng **Tailwind CSS**
* 🧩 Kiến trúc sạch và dễ mở rộng
* 🎬 Hiệu ứng mượt mà & giao diện tương tác
* 📱 Thiết kế responsive hoàn toàn
* 🖼 Tối ưu hóa tài nguyên và hiệu suất
* 🔗 Tích hợp mạng xã hội
* 📩 Phần liên hệ với nút kêu gọi hành động

---

## 📂 Cấu trúc dự án

```
app/            → Trang & định tuyến (Next.js App Router)
components/     → Các component UI tái sử dụng
config/         → Cấu hình chung (liên kết, cài đặt)
data/           → Dữ liệu tĩnh (mạng xã hội, nội dung)
lib/            → Tiện ích & hàm hỗ trợ
public/         → Tài nguyên tĩnh (hình ảnh, icon)
```

## 🚀 Bắt đầu

### 1. Clone repository

```bash
git clone https://github.com/your-username/portfolio.git
cd portfolio
```

### 2. Cài đặt dependencies

```bash
npm install
# hoặc
yarn install
# hoặc
bun install
```

### 3. Chạy server phát triển

```bash
npm run dev
```

### 4. Mở trên trình duyệt

```
http://localhost:3000
```

## ⚙️ Tùy chỉnh

Bạn có thể dễ dàng tùy chỉnh:

* ✏️ Nội dung văn bản → `data/`
* 🔗 Liên kết → `config/`
* 🎨 Giao diện → `components/`
* 🖼 Hình ảnh → `public/`
